商品情報管理システム
  • ログイン画面へ

はじめに

現在のところ、このシステムは単一ユーザー対応です。
ログインID「taro」、パスワード「abcd」でログインできます。

概要

商品の購入履歴やお気に入りをリスト形式で管理するシステム。
商品の情報は自分で登録する。
リンク先のURLを保存したり、登録したキーワードを使用した検索を行える。

開発経緯

商品を購入する際、実店舗や通販サイトを含む様々な店舗から、商品の価格などの情報を集めることが多いが、集めた情報を比較する場合に、一元的なリスト形式で見ることができれば便利だと思い作成した。
また、商品の情報を再度検索する場合に、検索に使用したキーワードを保存できれば検索の手間が省けると思った。
技術的には、データベースのCRUDを用いたデータベース管理システムを作成できることを示すため。

開発環境

OS Windows, Linux
言語 Java SE 11, HTML, CSS, JavaScript, JQuery 3.6.0
データベース MySQL
フレームワーク Bootstrap 4.6.0, Spring Boot 2.4.5, MyBatis 2.1.4
ツール Eclipse, MySQL Workbench 8.0 CE

苦労した点

  • 一括削除機能

    チェックを付けた項目を判別するための処理や、一つもチェックが付けられていない場合の判定処理、セルをクリックした場合でもチェックボックスの状態を変化させる処理など、この部分の実装に最も苦労した。

  • 登録フォームの入力値検証

    フォームに入力された値、特にリンク先URLに入力された値に対して、自分で決めたルールを適応させるためにはどうしたらいいのか、かなり試行錯誤した。

  • デザイン

    デザインについては全く触れて来なかったため、ひと目でわかりやすいシステムになっていないと思われる。Bootstrapを使用してはみたものの、なかなか思い通りには表示されなかった。
